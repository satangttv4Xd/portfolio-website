# Portfolio Website — Frontend + Backend

เว็บพอร์ตโฟลิโอที่แก้ไขเนื้อหา/อัปโหลดรูปได้จากหน้าเว็บโดยตรง ตกแต่งด้วยคอมโพเนนต์สไตล์ React Bits และดาวน์โหลดเป็น PDF ได้

**อ่านคู่มือฉบับเต็มภาษาไทยได้ที่ [`คู่มือการใช้งาน.md`](./คู่มือการใช้งาน.md)**

## เริ่มต้นอย่างเร็ว

```bash
# Terminal 1 — backend
cd backend
npm install
npm run dev      # http://localhost:5000

# Terminal 2 — frontend
cd frontend
npm install
npm run dev      # http://localhost:5173
```

เปิด `http://localhost:5173` ในเบราว์เซอร์ แล้วกดปุ่ม **"แก้ไขเว็บไซต์"** มุมขวาบนเพื่อเริ่มกรอกข้อมูลของคุณเอง

## ฟีเจอร์หลัก

- แก้ไขทุกส่วนของเนื้อหา (โปรไฟล์ เกี่ยวกับ ทักษะ ผลงาน ประสบการณ์ ติดต่อ) ผ่านแผงแก้ไขในหน้าเว็บ
- อัปโหลดรูปภาพ (โปรไฟล์ + ปกผลงาน) เก็บไว้ที่ backend ของคุณเอง
- ตกแต่งด้วยคอมโพเนนต์อนิเมชันสไตล์ React Bits: spotlight card, particle field, shiny button, scroll reveal
- ดาวน์โหลดพอร์ตโฟลิโอทั้งหน้าเป็นไฟล์ PDF ได้ในคลิกเดียว
- ข้อมูลทั้งหมดเก็บเป็นไฟล์ในเครื่อง ย้าย/สำรอง/กู้คืนได้ง่าย

## โครงสร้างโปรเจกต์

```
backend/    Express API + รับอัปโหลดรูป + เก็บข้อมูลเป็น JSON
frontend/   React + Vite + Tailwind CSS
```

รายละเอียดทุกขั้นตอน (ติดตั้ง ใช้งาน แก้ไขดีไซน์ deploy แก้ปัญหา) อยู่ใน [`คู่มือการใช้งาน.md`](./คู่มือการใช้งาน.md)
